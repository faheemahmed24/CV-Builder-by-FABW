# Contributing to CV Builder

First off, thank you for considering contributing to CV Builder! It's people like you that make CV Builder such a great tool.

## Code of Conduct

By participating in this project, you agree to abide by our Code of Conduct:
- Be respectful and inclusive.
- Focus on constructive feedback.
- Help others and share knowledge.

## How Can I Contribute?

### Reporting Bugs
- Use the [GitHub Issue Tracker](https://github.com/faheemahmed24/CV-Builder-by-FABW/issues).
- Describe the bug in detail, including steps to reproduce.
- Mention your browser and operating system.

### Suggesting Enhancements
- Use the [GitHub Issue Tracker](https://github.com/faheemahmed24/CV-Builder-by-FABW/issues).
- Explain why the enhancement would be useful.
- Provide examples or mockups if possible.

### Pull Requests
1. **Fork the repository** and create your branch from `main`.
2. **Branch naming convention**:
   - `feature/*` for new features.
   - `fix/*` for bug fixes.
   - `docs/*` for documentation changes.
   - `refactor/*` for code improvements.
3. **Commit message format**: `[TYPE] Description` (e.g., `[FEATURE] Add Urdu language support`).
4. **Coding standards**:
   - Use TypeScript for all new code.
   - Follow the existing project structure.
   - Ensure your code passes linting: `npm run lint`.
5. **Testing**:
   - If you're adding a feature, add tests if possible.
   - Ensure all existing tests pass: `npm run test`.
6. **Submit the PR**:
   - Provide a clear description of the changes.
   - Link any related issues.

## Development Setup

1. **Clone the repo**:
   ```bash
   git clone https://github.com/faheemahmed24/CV-Builder-by-FABW.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```
