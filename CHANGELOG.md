# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Enhanced documentation (README, CONTRIBUTING, ARCHITECTURE, USAGE)
- GitHub Actions workflows for Build & Test, Deploy to Pages, and Releases
- Internationalization (i18n) infrastructure with 10 languages
- RTL layout support for right-to-left languages
- Enhanced skills suggester and ATS scorer
- Shareable URLs for CV sharing
- Dark mode support
- Keyboard shortcuts

## [1.0.0] - 2026-03-28

### Added
- Initial release of CV Builder
- 3 professional templates (Classic, Modern, Minimal)
- 4 color themes (Blue, Gray, Green, Purple)
- Real-time preview with auto-save to localStorage
- Text improver for experience bullet points
- Skills suggester based on job title
- ATS scorer with improvement tips
- Export to PDF, JSON, and plain text
- 20-step undo/redo history
- Responsive design for mobile, tablet, and desktop
- Privacy-first, offline-first architecture
